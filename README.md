# comeon eslint config

Combines eslint with prettier

## Uses

```js
	npm install --save-dev @comeon/eslint-config
```

or 

```js
	yarn add --dev @comeon/eslint-config
```

## Add to eslint config

```js
// .eslintrc.js
module.exports = {
	extends: ['@comeon']
}

```
